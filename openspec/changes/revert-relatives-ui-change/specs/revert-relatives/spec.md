## ADDED Requirements

### Requirement: 恢复原版 Tabs 展示
系统 SHALL 恢复使用 van-tabs 组件展示亲戚分类，包含"爸爸这边的"、"妈妈这边的"、"伴侣这边的"三个标签页。

#### Scenario: 显示爸爸这边的亲戚
- **WHEN** 用户点击"爸爸这边的"标签
- **THEN** 显示爸爸一方的亲戚关系列表

#### Scenario: 显示妈妈这边的亲戚
- **WHEN** 用户点击"妈妈这边的"标签
- **THEN** 显示妈妈一方的亲戚关系列表

#### Scenario: 显示伴侣这边的亲戚
- **WHEN** 用户点击"伴侣这边的"标签
- **THEN** 显示伴侣一方的亲戚关系列表
