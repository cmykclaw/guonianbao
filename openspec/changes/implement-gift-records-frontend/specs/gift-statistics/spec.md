# Spec: gift-statistics

## ADDED Requirements

### Requirement: 统计卡片展示
页面顶部必须显示统计卡片，展示总收礼和总送礼金额。

#### Scenario: 统计卡片渲染
- **WHEN** 页面加载完成
- **THEN** 必须显示两个统计卡片
- **AND** 第一个卡片标题必须为"总收礼"
- **AND** 第二个卡片标题必须为"总送礼"

#### Scenario: 收礼金额计算
- **WHEN** 有收礼记录存在
- **THEN** "总收礼"卡片必须显示所有 RECEIVED 类型记录的金额总和
- **AND** 金额必须以"¥"符号开头
- **AND** 金额必须使用千分位格式化

#### Scenario: 送礼金额计算
- **WHEN** 有送礼记录存在
- **THEN** "总送礼"卡片必须显示所有 GIVEN 类型记录的金额总和
- **AND** 金额必须以"¥"符号开头
- **AND** 金额必须使用千分位格式化

#### Scenario: 空记录统计
- **WHEN** 没有任何记录
- **THEN** "总收礼"和"总送礼"都必须显示"¥0"

### Requirement: 统计卡片样式
统计卡片必须有明显的视觉区分。

#### Scenario: 收礼卡片样式
- **WHEN** 查看"总收礼"卡片
- **THEN** 金额文字必须为红色 (#ee0a24)
- **AND** 卡片必须有明显的收礼标识

#### Scenario: 送礼卡片样式
- **WHEN** 查看"总送礼"卡片
- **THEN** 金额文字必须为绿色 (#07c160)
- **AND** 卡片必须有明显的送礼标识
